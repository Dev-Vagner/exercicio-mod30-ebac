# sistema-de-vendas
Sistema de vendas em Java utilizando JDBC para conexão com o banco de dados. <br />
Utilizando:
- Testes unitários;
- Testes de integração;
- TDD;
- JDBC;
- SQL;
- PostgreSQL;
- Mock;
- Generics;
- Annotations;
- Reflections;
- Exceptions;
- Camada de Service e de DAO;
- Design Patterns.
